Valtteri Bottas

2019-11-13

"Valtteri, it's James."

I’ve been working on some art for a new project that expands the service our Server Manager provides for Assetto Corsa. Here’s Bottas.

This was my first time using an iPad (the iPad Pro 2018, that I borrowed) to draw something, and i’m pretty happy with how it came out! Mixing the “real” feeling of using a traditional pencil directly onto a drawing surface with all of the added benefits that come with working in art software really does create a powerful and unique experience.

So an iPad has been added to the wishlist, hopefully before the end of the year I’ll have one, I think I’ll get tons of use out of it for work.

Although if I’m being honest I just want to watch tv in bed with cookies, like a savage.
